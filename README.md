# Xscape Photonics — Newsletter Builder

A self-contained, single-file newsletter template builder tool. Open `index.html` in a browser to compose, edit, and export branded email newsletters.

## Usage

- **Local:** open `index.html` in any modern browser.
- **Deployed:** served as a static site (Vercel).

The entire app (markup, styles, fonts, and logic) lives in `index.html` — no build step or dependencies.
